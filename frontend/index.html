Commands

Everything you need,
in the terminal.

cvz analyze
Score a resume against a specific role or custom job description. Auto-detects the best role if none is provided.
cvz analyze resume.pdf --role backend
cvz gap
Show exactly which core skills are present and which are missing for a given role. Export as JSON.
cvz gap resume.pdf --role machine_learning
cvz compare
Rank multiple resumes side by side. Useful for hiring or self-benchmarking across different versions.
cvz compare r1.pdf r2.pdf r3.pdf
cvz suggest-role
Auto-detect the top matching roles from the resume without specifying one. Shows ranked results.
cvz suggest-role resume.pdf --top 10
cvz roles
List all supported roles. From backend to blockchain, devops to UI/UX — 20+ roles available.
cvz roles
--export
Export any report to a JSON file. Works with analyze, gap, and compare commands.
cvz analyze resume.pdf --export report.json
How It Works

Three engines.
One honest score.

cvz doesn't rely on a single metric. Every resume is evaluated across three independent dimensions — keyword coverage, semantic similarity via spaCy word vectors, and structural completeness.

The final score is a weighted average. Semantic scoring is automatically gated — if keyword coverage is too weak, semantic weight is reduced to prevent vector-level noise from inflating the result.

Great match ≥ 75%
Decent match 50 – 74%
Low match < 50%
Keyword Matching 50%
Core (70pt) + Secondary (30pt) keyword groups. Whole-word matching.
Semantic Similarity 30%
spaCy cosine similarity. Optional — falls back gracefully if model is absent.
Structure Analysis 20%
Weighted section presence (Education, Experience, Skills…) + resume length.
